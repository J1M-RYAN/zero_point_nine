# Zero Point Nine

A cli that allows you to send HTTP/0.9 requests.

## Installation

### From Crates.io
```bash
cargo install zero_point_nine
```

### From Repository
```bash
git clone https://github.com/J1M-RYAN/zero_point_nine.git
cd zero_point_nine
cargo install --path .
```

## Usage

```bash
zero_point_nine localhost 8080 /
```
